# Mini Information Retrieval System

## Overview

This project is a learning implementation of an Information Retrieval (IR) system in Python.

The current version supports simple keyword-based document ranking. User queries are matched against a collection of documents, and documents are ranked based on the number of matching query terms.

## Features

* Query input from the terminal
* Document tokenization
* Keyword matching
* Document ranking and sorting

## Example

Documents:

* python flask web framework
* python programming language
* football sport game

Query:

python framework

Result:

* python flask web framework
* python programming language
* football sport game

## Current Status

This is an early prototype built to understand the fundamentals of Information Retrieval.
